uieComponent

description

Type: component's type

The entire component's description.

usage

var myComponent = new uieComponent(id,options);

id

String

The ID attribute of the newly created DOM element.

options

Object

A collection of properties for customize the component's layout and behaviors.

properties

You can pass some (or all) of these properties within an object inside constructor.

(Object) extension

Css extension object for customizable elements.

Default: {}

Properties accepted:


   myFirstElement
   mySecondElement

methods

methodName(param)

returns

The method's full description.

example


   var myExtension = {
      myClass : {position:'absolute','text-align':'right'}
   };
   var myOptions = {
      extension:    myExtension
   };
   var myComponent = new uieComponent('myComponentID',myOptions);
   $('body').append(myComponent);

   myComponent.publicMethod();
